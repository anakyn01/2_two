statement : 자바스크립트 문

- 컴퓨터 프로그램은 컴퓨터에 의해 실행되는 "명령"의 목록입니다
- 프로그래밍 언어에서는 이러한 프로그래밍 지침을 명령문이라고 합니다
- javascript프로그램은 프로그래밍 명령문의 목록 입니다
- 문은 다음으로 구성됩니다 값,연산자,표현식,키워드 및 설명
- 문은 우리가 일반적으로 알고 있는 용어인 javascript코드라 합니다
- 세미콜론은 자바스크립트 문을 구분합니다;(한줄에 여러문이 허용됩니다)

자바스크립트 키워드

var = 변수를 선언
let = 재정의,재할당 금지에 변수 선언
const = 상수(변하지 않는 변수선언)를 선언
if = 조건에 따라 실행될 명령문 표시
switch = 다양한 경우에 실행될 명령문 블록을 표시
for = 루프(지정된 범위 까지 반복)를 실행
function = 함수를 선언
return = 함수 종료
try = 명령문에 대한 오류 처리 구현

자바스크립트 세미콜론

값

- 자바스크립트 구문은 두가지 유형에 값을 정의합니다
고정값 = 리터럴
1. 숫자는 소수점 유무에 관계없이 작성됩니다
10.50
1001

2.문자열은 큰따옴표나 작은 따옴표로 작성된 텍스트 입니다
"dlfma"
'이름'

변수값 = 변수는 데이터 값을 저장하는데 사용됩니다
자바스크립트는 변수를 선언하기 위해서 var,let,const라는 키워드를 사용합니다
등호는 변수에 값을 할당하는데 사용됩니다
아래에 예에서는 x가 변수로 정의됩니다 그런다음에 x에 6이 할당됩니다
let x; //변수 선언
x = 6; //x에 값을 할당
