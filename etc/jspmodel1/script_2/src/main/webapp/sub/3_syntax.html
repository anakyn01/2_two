syntax : 구문

- javascript 구문은 javascript 프로그램이 구성되는 방식에 대한 규칙 집합입니다.
- 자바스크립트 값 두 가지 유형의 값을 정의합니다
고정값[리터럴], 변수값[변수]
- 변수란 무엇인가요 변하는 수여서 변수, 많은수여서 변수
- 리터럴 (고정값)은 두가지 규칙이 있습니다
숫자는 소수점 유무에 관계없이 작성됩니다
문자열은 큰 따옴표나 작은 따옴표로 작성됩니다
- 변수 : 데이터 값을 저장하는데 사용됩니다
- 변수 키워드 var,let,const =(할당 연산자를 사용하여) 값을 대입합니다
- 자바스크립트는 대소문자를 구분합니다
- Camel case : 변수명을 사용하는 방법 CamelCase
- snake Case : snake_case
- Lower Camel Case : firstName
