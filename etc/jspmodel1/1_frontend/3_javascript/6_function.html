함수

- 함수는 특정 작업을 수행하도록 설계된 코드 블록입니다
- 함수명으로 호출할때 실행됩니다
- 자기스스로 값을 나타내는걸 재귀함수
- 문법 
function 함수명(){
    코드블럭 안에 내용
}    
-함수명(안에 소괄호에 매개변수는 사용할수 있다)
function name(parameter1, parameter2)
- 함수를 호출하는 경우
이벤트가 발생했을때 (사용자가 버튼을 클릭했을때)
자바스크립트 코드에서 호출되는 경우
자동으로 자체 호출
- 왜 함수를 사용하나요?
함수를 사용하면 코드를 재사용(여러번 사용할수 있는 코드를 작성) 할수 있습니다
동일한 코드를 다른 인수와 함께 사용하여 다른 결과를 생성할수 있습니다
