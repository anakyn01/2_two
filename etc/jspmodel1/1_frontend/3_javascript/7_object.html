class와 객체

class : 객체를 만들기 위한 템플릿
object : class에서 필드나 메서드를 상속받아 사용
그렇다면 객체는 무엇인가요?
java, javascript,react,vue,python,nodejs,nextjs
전부 객체지향 언어입니다
원시값(primitive 더이상 단순화 될수 없는 기본자료형)을 
제외한 모든것이 객체
string,number,bigint,boolean,undefined,symbol,null
자바스크립트에서 객체는 실제생활과 같은
집,자동차,사람,동물 또는 기타 주제와 같습니다
Nintendo(class), wii, superfamicom, 
현대(class), 아반떼(객체)
객체는 속성(메소드)을 가지고 있습니다 
(name, model=2024, 무게:850kg, 색상:블랙)
