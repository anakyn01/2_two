변수

변수는 데이터를 저장하는 컨테이너 입니다
변수는 4가지 방법으로 선언할수 있습니다
- 자동으로

x = 5;
y = 6;    

- var : 1995년 부터 2015년까지 사용

var x = 5;
//선언하고 값을 대입
var x;
x = 5;   

- let : ES6(2015)에서 도입
사용하기 전에 선언되어야 합니다, 선언된 변수는 다시 선언 될수 없다
재선언,재할당 금지
블록내부에서 선언된 변수는 외부에서 엑세스 할수 없습니다
똑같은 변수를 블록안에서 선언하면 출력은 외부가 되지만 선언할수 있다
- const : ES6(2015)에 도입 되었습니다
상수 변하지 않는 참조만 하는 수
