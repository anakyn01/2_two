자바스크립트 배열

- 배열은 하나 이상의 값을 보유할수 있는 특수 변수입니다
- 왜 배열을 사용하는가 ?
단일 변수명으로 여러값을 저장할수 있는 편리한 솔루션이기 때문에
- syntax
const array_name = [item1, item2...]
- array 만드는 2가지 방법
(1. 정해진 값을 사용, 2.new Array를 사용해서 생성)
- 배열값에 엑세스 : 색인을 사용(자바, 자바스크립트는 0이 첫번째)
- 색인을 이용해서 내용을 변경할수 있다
- for를 사용하여 반복자 [(iterator) 지정된 범위까지 출력]로
